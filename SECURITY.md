# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in KafkaCode, please report it
responsibly. **Do not open a public GitHub issue for security problems.**

Email: **contact@kafkalabs.com**

Please include:

- A description of the vulnerability and its potential impact
- Steps to reproduce
- Any suggested remediation, if you have one

We aim to acknowledge reports within 72 hours and will keep you informed as we
investigate and ship a fix.

## Supported Versions

The latest published `1.x` release receives security updates.
