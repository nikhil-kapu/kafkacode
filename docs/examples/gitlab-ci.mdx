## Basic GitLab CI Pipeline

Create `.gitlab-ci.yml`:

```yaml
stages:
  - security

privacy-scan:
  stage: security
  image: node:18
  before_script:
    - npm install -g kafkacode
  script:
    - kafkacode scan ./src --verbose
  allow_failure: false
```

## For more examples and advanced configurations, see the [CI/CD Integration Guide](/usage/ci-cd-integration).
